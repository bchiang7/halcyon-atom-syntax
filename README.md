## Installation

### Via Preferences

1. Go to `Preferences > Install` and choose `Themes`
2. Search for `halcyon-syntax` and install it
3. Go to `Preferences > Themes` and choose Halcyon as the syntax theme

### Via APM

`apm install halcyon-syntax`
